# 花园回响 EchoBloom

当前实现的是低保真页面结构和基础交互逻辑，暂不包含最终视觉设计。

## 页面

- `index.html`：首页，包含「进入我的花园」和「与花灵聊聊」。
- `chat.html`：花灵聊天页。
- `flower.html`：今日花朵生成页。
- `garden.html`：花园主页。

## 交互

- 点击首页「进入我的花园」跳转到 `garden.html`。
- 点击首页「与花灵聊聊」跳转到 `chat.html`。
- 在花灵聊天页发送内容后，会出现「生成今日花朵」按钮，点击跳转到 `flower.html`。
- 在花朵生成页点击「放入花园」跳转到 `garden.html`。
- 花园主页左侧栏上方依次是「花园回响」和「今日花事」。
- 点击「今日花事」后，今日花事内容显示在左侧栏内，右侧主区域暂时留给花圃。

## 本地预览

直接用浏览器打开 `index.html` 即可。
